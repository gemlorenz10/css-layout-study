# css-layout-study
Basic css layout study - made for educational purposes only.

## Run
```
$npm i
$npm run serve
```
## Change CSS file into either float or  flex
```HTML
    <!-- Choose display layout -->
    <link rel="stylesheet" href="float/style.css">
    <!-- Or -->
    <!-- <link rel="stylesheet" href="flex/style.css"> -->
```
